📖 วิธีสร้าง Firebase ฟรี
1
ไปที่ console.firebase.google.com แล้ว สร้าง Project
2
เลือก Realtime Database → Create Database → เลือก Location ใกล้ที่สุด
3
เลือก Start in test mode (สำหรับใช้งานทั่วไป) → Enable
4
คัดลอก Database URL ที่ขึ้นต้นด้วย https:// วางในช่องด้านบน
5
กด 💾 บันทึก URL แล้วระบบจะซิงค์ข้อมูลอัตโนมัติทุก 15 วินาที
💡 หมายเหตุ: ทุกเครื่องที่ใส่ URL เดียวกัน จะแชร์ข้อมูลกัน Real-time — สำหรับทีมก่อสร้างสามารถเปิดพร้อมกันได้หลายเครื่อง
