สร้าง Firebase ฟรี
Real-time Sync สำหรับโครงการ
1
เปิด Firebase Console
ไปที่ console.firebase.google.com → Add project → ตั้งชื่อโครงการ
2
สร้าง Realtime Database
เมนูซ้าย → Build → Realtime Database → Create Database → เลือก Location (asia-southeast1 สำหรับไทย)
3
เลือก Test Mode
เลือก Start in test mode → Enable — ข้อมูลจะ Public อ่าน/เขียนได้ 30 วัน (ต่ออายุได้)
4
คัดลอก Database URL
หลัง Create เสร็จ จะเห็น URL ในรูปแบบ:
https://your-project-default-rtdb.asia-southeast1.firebasedatabase.app
คัดลอก URL นี้ → วางในช่อง Firebase URL ในฟอร์ม
5
ทดสอบและบันทึก
วาง URL แล้วกด 🔌 ทดสอบ เพื่อยืนยันการเชื่อมต่อ → กด ✅ บันทึกโครงการ
💡 เคล็ดลับ: สร้าง Firebase Project แยกสำหรับแต่ละโครงการก่อสร้าง เพื่อให้ข้อมูลไม่ปนกัน และทุกเครื่องที่ใส่ URL เดียวกันจะ Sync ข้อมูล Real-time
